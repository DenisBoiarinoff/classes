# Python - Изучение классов

## Описание: 

Проект для изучения классов

## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/DenisBoiarinoff/study_bank_operations_widget.git
```

2. Установите зависимости:
```
poetry install
```
